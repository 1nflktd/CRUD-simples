# CRUD-Simples

- Banco de dados:

	```sql
	CREATE DATABASE crud_simples;
	```

- Tabela a ser criada:

	```sql
	CREATE TABLE `filme` (
		`id` INT AUTO_INCREMENT,
		`nome` VARCHAR(100) NULL,
		`ano` INT NULL,
		PRIMARY KEY (`id`)
	);
	```

- Para conectar no database dentro do container:

	```
	mysql -u docker_user -P 3307 -h 127.0.0.1 -p
	```
